# Tatva Dynamics (MEPL) — Precision Manufacturing

A high-authority, industrial-grade web presence for **Tatva Dynamics**, a premier manufacturing partner specializing in 5-axis CNC machining, precision sheet metal fabrication, and complex industrial assemblies.

## 🚀 "Midnight Engineering" Design System

The application is built using the **Midnight Engineering** aesthetic—a cinematic, high-contrast B2B design language characterized by:
- **Pure Black (#000000) Backgrounds** for zero-compromise authority.
- **Electric Accent Lighting** using HSL-tailored primary blues.
- **Dynamic Grid Visuals** and electrical pulse animations.
- **Industrial Typography** featuring Inter and IBM Plex Mono.

## 🛠 Technology Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Smooth Scroll:** Lenis
- **AI Integration:** Google Gemma 4 31B (via OpenRouter)

## 📁 Key Features

- **Cinematic Hero:** Immersive video loop with soft-edge blending.
- **AI Assistant:** Integrated chatbot with a 60-point industrial FAQ knowledge base.
- **Strategic Locations:** Interactive maps and facility panoramic visuals with lightboxes.
- **Product Catalog:** High-fidelity technical grid with detail modals.
- **Defense-Ready:** Specialized sections for high-authority defense and aerospace manufacturing.
- **RFQ System:** Streamlined Request for Quote flow with secure validation.

## 🏁 Getting Started

### 1. Installation
```bash
npm install
```

### 2. Environment Setup
Create a `.env.local` file with the following:
```env
# Contact Form (Resend)
RESEND_API_KEY=re_...
RESEND_FROM_EMAIL=...
RESEND_TO_EMAIL=...

# AI Assistant (OpenRouter)
OPENROUTER_API_KEY=sk-or-v1-...
```

### 3. Run Development Server
```bash
npm run dev
```

## 🌐 Deployment

The project is optimized for deployment on **Vercel**. 
1. Push the code to a GitHub repository.
2. Import the project in Vercel.
3. Configure Environment Variables in the Vercel Dashboard.
